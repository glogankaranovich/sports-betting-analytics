# Frontend

> React/TypeScript app hosted on AWS Amplify.

## Directory
`frontend/src/`

## Stack
- React with TypeScript
- AWS Amplify for hosting and auth
- Cognito for user authentication

## Features
- Game analysis dashboard
- Bet tracking and history
- Model performance comparison (V1 vs V3)
- AI chat agent for strategy questions
- User-defined model configuration

## Deployment
- Amplify auto-deploys from main branch
- Config: `frontend/amplify.yml`
